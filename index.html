PROGRAM 01

마인크래프트 속 용산 메타버스

우리가 살고 있는 용산을 마인크래프트 안에서 다시 짓는 도시 탐험 수업. 지도와 사진을 보고 랜드마크를 모델링하며, 친구들과 함께 메타버스를 완성합니다.

  • 지도 분석 → 마크 좌표 설계
  • 랜드마크 프로그래밍 빌드
  • 가상 용산 투어 발표

PROGRAM 02

용산 스마트 시티 보안관 제작하기

AI를 활용해 용산을 더 안전하게 만드는 스마트 시티 보안관을 직접 설계. 데이터를 학습시키고, 위험을 알아채는 모델을 만들어 우리 동네에 적용해 봅니다.

  • 소리 데이터 수집
  • 빛센서, 라인트레이싱 이용
